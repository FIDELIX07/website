  1. Home Technologies
 LiveData
LiveData allows me to follow the observer pattern. LiveData notifies Observer objects when the lifecycle state changes. Instead of updating the UI every time the app data changes, your observer can update the UI every time there's a change. Its provided by Google hence well designed.
 Firebase Realtime Database
is a cloud-hosted NoSQL database that enables data to be stored and synced between users in real time. The data is synced across all clients in real time and is still available when an app goes offline. This has no close competitor.
 Google Maps
Android allows us to integrate google maps in our application. You can show any location on the map , or can show different routes on the map e.t.c. You can also customize the map according to your choices. It is the most well delevoped SDK for mapping by Google.
 Firebase Auth
Firebase auth has a built in email/password authentication system. It also supports OAuth2 for Google, Facebook, Twitter and GitHub. We’ll focus on email/password authentication for the most part. Firebase’s OAuth2 system is well-documented. This is secure since it is used even by other technologies for OAuth2.
 Crashlytics
Firebase Crashlytics is a real-time crash reporter that will help me to track, to prioritize and to fix stability issues that reduce the quality of this app. This is so effective that the Google Play Store endorses it.will aid me to spend less time organizing and troubleshooting crashes and more time building features.
 Analytics
Google Analytics for Firebase offers free, unlimited reporting on as many as 500 separate events. Analytics presents data about user behavior in iOS and Android apps, enabling better decision-making about improving performance and app marketing. It has been used for long hence has a history of reliability.
